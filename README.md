# GoogleForms

A Visual Basic .NET project inspired by Google Forms for building and collecting survey responses.

## What it does
- Create form definitions with questions and options
- Collect responses from users
- View submitted results in a clean interface

## Tech Stack
- Visual Basic .NET
- Windows Forms / desktop UI

## Project Structure
```
├── README.md
├── sources/
└── project files
```

## Getting Started
1. Open the solution in Visual Studio
2. Build and run the application
3. Create a form and begin collecting responses
